# Snake_water_gun-
A beginner-friendly Python project that plays the Snake-Water-Gun game with the user using simple conditional logic.

Snake Water Gun – Python Project
This is a simple Python program that lets the user play the classic Snake–Water–Gun game against the computer.

Rules:
Snake (1) drinks water → Snake wins
Water (0) douses gun → Water wins
Gun (-1) shoots snake → Gun wins
Same choices → Draw

Technologies Used:
Python
Conditional statements
User input

How to Run:
1. Install Python 3
2. Save the file as snake_water_gun.py
3. Run the program:

python snake_water_gun.py
